Sphinx在Windows下环境配置与使用

sphinx

1.官网下载任意解压包；
2.解压剪切至d:\sphinx，并在d:\sphinx新建目录data与log；

3.将d:\sphinx\sphinx.conf.in复制到d:\sphinx\bin\sphinx.conf.in，并重命名为sphinx.conf
4.修改sphinx.conf
4.1 将@CONFDIR@替换成实际目录
4.2 修改source src1

4.3 修改index test1

5.建好测试数据库
6.建索引（PS:先建索引，后安装服务，管理员权限）

7.安装服务
searchd.exe --install -c d:\sphinx\bin\sphinx.conf --servicename SphinxSearch
PS:config路径务必 写好全称
net start SphinxSearch (启动服务)
sc delete SphinxSearch （删除服务）
win+R services.msc （打开服务面板）
8.java API make 产生jar包
9.设置最大显示记录条数
SphinxClient sphinxClient; sphinxClient.SetLimits(offset, limit, maxMatch);
