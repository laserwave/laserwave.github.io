黑白棋游戏规则

初始状态

游戏棋盘为8X8，初始状态为两枚白子与两枚黑子交错位于棋盘中央

落子条件

落子位置必须为空格，并且能够翻转敌方棋子

翻转条件

落子与本方棋子在横、竖、正斜、反斜各方向将敌方棋子夹住

轮空规则

当某方无处可落子时则轮空，由敌方继续下子

游戏结束条件

某方无子或双方均无处落子

胜负判定

达到游戏结束条件之时，棋子多者获胜，少者失败，若相等则为平局

Rule类实现游戏规则相关的方法，包括

1 判断某一步是否合法

Algorithm类实现极小极大算法，包括

1 局面评估函数，对当前局面打分，越高对max越有利，越低对min越有利
2 min()方法
3 max()方法

ReversiView实现游戏界面

ReversiView继承自SurfaceView实现界面，在该类定义游戏界面的绘制、更新等操作

RenderThread控制ReversiView

RenderThread继承自Thread，控制ReversiView以一定fps更新、重绘界面，实现动画效果

游戏算法类Algorithm暨极小极大算法的实现

主要看一下max和min方法，两个方法的参数均为byte[][] chessBoard, int depth, int alpha, int beta, byte chessColor, int difficulty
chessBoard为棋盘；
depth为博弈树深度；
alpha和beta用于剪枝，在max方法中alpha一直取评估值的最大值(初始化为负无穷)，在min方法中beta一直取评估值的最小值(初始化为正无穷)，
当alpha >= beta时就进行剪枝；
chessColor表示当前方的棋子颜色；
difficulty表示游戏难度
